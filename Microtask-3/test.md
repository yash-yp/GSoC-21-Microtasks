## Code Formatting
```python
# Function for nth Fibonacci number
def fib(n):
	if n < 0:
		print("Enter non-negative number")	
	elif n == 0:
		return 0
	elif n == 1 or n == 2:
		return 1
	else:
		return fib(n-1) + fib(n-2)

# Driver Code
print(fib(9))

```
